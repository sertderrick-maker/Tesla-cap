API Documentation

Build with TeslasCap. Integrate our powerful investment and trading APIs into your application.

Getting Started

The TeslasCap API allows developers to access investment data, execute trades, and manage portfolios programmatically. All API requests require authentication using an API key.

Authentication

Include your API key in the Authorization header of all requests:

Authorization: Bearer YOUR_API_KEY

Base URL

https://api.teslascap.com/v1

Endpoints

Accounts

GET
/accounts/{account_id}

Retrieve account information including balance, holdings, and performance metrics.

GET
/accounts/{account_id}/positions

Get all current positions in an account with real-time valuations.

Trading

POST
/orders

Create a new buy or sell order for stocks or crypto.

GET
/orders/{order_id}

Get the status and details of a specific order.

DELETE
/orders/{order_id}

Cancel an open order.

Market Data

GET
/quotes/{symbol}

Get real-time quote data for a stock or crypto asset.

GET
/bars/{symbol}

Retrieve historical price bars (OHLCV data) for technical analysis.

Portfolios

GET
/portfolios

List all portfolios associated with your account.

POST
/portfolios

Create a new investment portfolio with custom allocation.

Rate Limits

API requests are limited to 1,000 requests per minute per API key. Exceeding this limit will result in a 429 (Too Many Requests) response.

Error Handling

All errors are returned as JSON with an appropriate HTTP status code and error message:

{ "error": "Invalid API key", "status": 401 }

Webhooks

Subscribe to real-time events like order fills, position changes, and market alerts. Configure webhooks in your account settings.
